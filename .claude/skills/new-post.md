# new-post

创建新的 Hugo 博客文章。

## 使用场景

当用户想要新建一篇博客文章时使用。用户会提供文章标题，可选提供标签和摘要。

## 调用方式

```
/new-post 文章标题
```

## 执行步骤

### 1. 解析用户输入

从用户输入中提取：
- **标题**（必需）：文章标题
- 如果用户没有提供标题，询问用户

### 2. 创建 Page Bundle

在 `content/post/{标题}/` 下创建目录和 `index.md` 文件。

目录名直接使用文章标题（中文即可，与现有文章风格保持一致）。

### 3. 生成 Front Matter

使用以下模板生成 `index.md`，格式与现有文章保持一致：

```markdown
---
title: "{标题}"
date: {当前时间，格式 2006-01-02T15:04:05+08:00}
url: "/{年}/{月}/{日}/{标题}/"
tags:
  - "{标签}"
draft: true
---

在这里开始写作...
```

注意事项：
- `date` 使用当前时间，时区 +08:00
- `url` 按照 `/{YYYY}/{MM}/{DD}/{标题}/` 格式生成，与旧文章 URL 风格一致
- `draft: true` 默认为草稿，写完后改为 false 再发布
- 如果用户提供了标签就使用，没有就先不加 tags 字段

### 4. 提示用户

创建完成后告诉用户：
- 文件路径
- 本地预览命令：`hugo server -D`（`-D` 显示草稿）
- 写完后记得把 `draft: true` 改为 `draft: false`
- 如果有封面图，放一张 `feature.jpg` 或 `feature.png` 到同目录下
- 文章中引用图片用相对路径，如 `![描述](image.jpg)`，图片文件放同目录
