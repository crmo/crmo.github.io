## Project Overview

This is the **GitHub Pages deployment repository** for a Hexo-powered personal tech blog (crmo.github.io). It contains **compiled static output only** — not the Hexo source files. The blog focuses on iOS development topics and is written in Chinese (zh-Hans).

Theme: NexT v6.0.4 (Muse scheme)

## Important: This Is a Build Output Repository

- There is no `package.json`, `_config.yml`, or `Gemfile` — source files live in a separate Hexo project
- All HTML/CSS/JS files are generated by `hexo generate` and pushed here for GitHub Pages hosting
- Commits follow the pattern "Site updated: YYYY-MM-DD HH:MM:SS"
- Direct edits to HTML files will be overwritten on next Hexo deploy

## Repository Structure

- `2016/`, `2018/`, `2019/`, `2020/` — Blog posts organized by year, each post at `/YYYY/MM/DD/{slug}/`
- `archives/` — Archive listing pages
- `tags/` — Tag-based category pages (iOS知识小结, 源码学习, 隐私政策, 其它)
- `page/` — Paginated post listings
- `images/` — Image assets referenced by posts
- `css/main.css` — NexT theme compiled stylesheet
- `js/src/` — Theme JavaScript (motion, scrollspy, search, sidebar, etc.)
- `lib/` — Vendored libraries (Font Awesome 4.6.2, jQuery, Velocity.js, ua-parser-js)

## SEO and Integration Files

- `atom.xml` — RSS/Atom feed
- `search.xml` — Local search index
- `sitemap.xml` / `baidusitemap.xml` — Search engine sitemaps
- `ads.txt` / `app-ads.txt` — Google AdSense config (pub-2365937932734432)
- `robots.txt` — Crawler directives
- `googlec6f436d36ac5bb4e.html` — Google Search Console verification
